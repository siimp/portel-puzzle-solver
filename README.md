## What are the number of possible combinations?

Possible rotations for 3x3 board is 4<sup>9</sup>  
Possible permutations for board P<sub>9</sub>  = 9!

P<sub>9</sub> * 4<sup>9</sup> = 9! * 4<sup>9</sup> = 95126814720
